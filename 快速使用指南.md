# 快速使用指南

## ✅ 项目已就绪

你的文件管理系统已经完全配置好，包含以下功能：

### 1️⃣ 多用户支持 ✅
- ✅ 每个用户独立的文件目录
- ✅ 自动支持并发操作（无需配置）
- ✅ 管理员可以切换查看其他用户文件（使用 UserSwitcher 组件）

### 2️⃣ 独立文件管理页面 ✅
- ✅ 路由：`http://localhost:3003/files-only`
- ✅ 全屏模式，无左侧聊天界面
- ✅ 可以直接在你的项目中使用

## 🚀 立即开始

### 方式一：快速测试（1分钟）

```bash
# 1. 启动服务
cd /Users/tezign/Documents/file-manager
pnpm dev

# 2. 访问独立文件管理页面
open http://localhost:3003/files-only
```

### 方式二：集成到你的项目

在你的现有 Next.js 项目中添加按钮：

```tsx
// 你的页面.tsx
'use client';

export default function YourPage() {
  const handleOpenFiles = () => {
    // 新窗口打开文件管理
    window.open('http://localhost:3003/files-only', '_blank');
  };

  return (
    <div>
      <h1>你的应用</h1>
      <button onClick={handleOpenFiles}>
        📁 管理文件
      </button>
    </div>
  );
}
```

## ❓ 常见问题

### Q: 如何增加文件大小限制？
A: 修改 `.env.local` 中的 `MAX_FILE_SIZE=104857600`（字节）

### Q: 如何添加新用户？
A: 修改 `lib/auth.ts` 中的 `initUsers()` 函数

### Q: 多用户会冲突吗？
A: 不会，每个用户有独立目录，自动支持并发

### Q: 管理员如何查看其他用户文件？
A: 在 FileManager 中添加 `UserSwitcher` 组件
